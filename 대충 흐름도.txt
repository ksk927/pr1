전체 구조 

사용자의 요청 -> 요청에대한 객체 생성(Action) -> 
Action의 메서드 호출 
-> Console Util 모듈 객체 생성 및 사용자의 입력 받기 
-> Service 객체 생성 및 Service 객체의 메서드 실행 



-> Service 객체의 메서드 내에서 DAO객체 생성 및 DAO 객체의 메서드 실행 
-> DAO객체의 메서드에서 데이터(List)에 접근하여 데이터 관련된 처리 진행 -> 
Service로 요청에 대한 처리실패 유무 피드백 -> 
서비스 에서 Action으로 처리실패 유무 피드백 -> 
UI system console 출력으로 피드백
